# CogniFace

AI-powered facial recognition attendance management system for educational institution.

## Features

- Real-time facial recognition attendance
- Administrator dashboard
- Lecturer dashboard
- Student dashboard
- Attendance analytics
- Course and enrollment management
- Secure authentication and authorization

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Material UI

## Installation

```bash
git clone https://github.com/tejasweebhetwal1/CogniFace-Project.git
cd CogniFace-Project
npm install
npm run dev
